# KC85/KC87-Linkliste
Eine Übersicht aller nennenswerten Webseiten und Projekte rund um den Robotron KC85 und KC87.

Welche Seiten fehlen? Welche Projekte findet Ihr spannend? 
<br>Ich freue mich über Vorschläge...




## Deutschsprachige Webseiten

| Webseite | Neue Zusammenfassung |
|----------|----------------------|
| [KC‑Club](https://www.kcclub.de) | Aktive Vereinsseite mit Infos zum KC‑Club, KC‑News, Treffen, Downloads und Archiv der alten Homepage. |
| [KC85.INFO](http://kc85.info/index.php) | Umfangreiche, gepflegte Seite mit Projekten, Software, CAOS/CP/M‑Tools, Systemprogrammen und Dokumentation zum KC85. |
| [Der KC85/4](https://www.mpm-kc85.de/) | Umfangreiche technische Dokumentation, Module, Zubehör, CAOS, MicroDOS, Animationen, Literatur |
| [Back to die Urzeit / KC85EMU](http://www.kc85emu.de/) | Emulator‑Seite von Frank & Tomas Ludwig, mit KC85EMU, Software, Dokumenten und Rückblicken. |
| [KCemu (SourceForge)](http://kcemu.sourceforge.net/) | KC‑Emulator von Torsten Paul, unterstützt KC85‑Reihe, Z1013, LC80 u.a.; inkl. D004‑Emulation und virtueller HD. |
| [JKCEMU](http://www.jens-mueller.org/jkcemu/index.html) | Moderner, sehr umfangreicher Multi‑Plattform‑Emulator für DDR‑Kleincomputer; aktiv gepflegt. |
| [robotrontechnik.de](http://www.robotrontechnik.de/) | Großes virtuelles Museum zur DDR‑Rechentechnik, mit Forum, Dokumentation, Bildern und Hintergrundwissen. |
| [Ulrich Zander](http://www.sax.de/~zander/) | Umfangreiche technische Infos, Schaltbilder, Programme und Projekte zu Z9001, KC85/1, KC87, BIC, A5105 u.a. |
| [Informatik in der DDR](http://waste.informatik.hu-berlin.de/Diplom/robotron/studienarbeit/files/frames.html) | Historische Studienarbeit über Informatik und Rechentechnik in der DDR; technisch veraltete Seite. |
| [DDR‑Rechentechnik](http://www.ddr-rechentechnik.de/) | Private Seite über Entwicklung der DDR‑Heimcomputer, CPU U880, Z1013, LC80, Robotron‑Rechner. |
| [Homecomputer DDR](https://hc-ddr.hucki.net/wiki/doku.php) | Sehr aktive, umfangreiche Wissenssammlung von Volker Pohlers zu Z9001/KC87, LC80, AC1, Z1013 u.v.m. |
| [KC85‑Museum](http://www.kc85-museum.de/) | Dokumentation der Rechner und Module, Schaltpläne und Software ; vermutlich nicht mehr gepflegt. |
| [Mike McBike Museum](https://www.wolfgangrobel.de/museum/kc853.htm) | Umfangreiche Informationen zu Elektronik, Microcontrollern, FPGAs, Arcade, Emulatoren, RC5, Programmierung, SR-500, Röhren und auch robotron Rechnern. |   
| [Wikipedia: Robotron Z9001 / KC85/1 / KC87](https://de.wikipedia.org/wiki/Robotron_Z9001) | Technische Daten, Historie, Hardware‑Specs, Produktionsdaten |
